Firmware Update

Update your Sendex vest to the latest firmware via USB.

Current version

Firmware v1.2.3 latest

Latest improvements: GPS stability, BLE reconnection, battery reporting.

How to update

1

Connect the Sendex vest to your computer via USB cable.Use a data-capable USB cable (not just charging).

2

Put the vest in update mode.Hold the button for 5 seconds until the LED blinks rapidly, or press the reset button twice.

3

Click the button below, select "ESP32" or "USB Serial" when prompted, and wait.

Web Serial not supported. Use Chrome or Edge on desktop. Please connect the vest via USB first.
⏱ Takes about 30 seconds. Do not unplug the USB cable during flashing.
USB Serial or CP2102 should appear in the browser prompt.

Manual update (alternative)

If the web installer doesn't work, you can flash using Python:

1

Install esptool: pip install esptool

2

Download the firmware: sendex-firmware-v1.2.3.bin

3

Run: esptool.py --chip esp32 --port COM3 write_flash 0x0 sendex-firmware-v1.2.3.bin
